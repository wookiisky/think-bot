# 洞察助手 (Think Bot)

## 项目简介
这个浏览器扩展是一个网页内容提取+预设提示词+大模型调用+阅读/对话的工具。是我在构建认知提升流程中迭代开发的，目标是减少深度阅读遇到的各种摩擦，目标是一个顺手好用的工具，也是我的在认知提升方面理念的体现。

](https://mp.weixin.qq.com/s/ASHxC4NQBBBrqB8zNK_dHA)
- [我的 AI 认知提升系统2：认知助手、框架和使用原则](https://mp.weixin.qq.com/s/lEBbcDZnts6Qh81LDR5iPQ)

## 关于代码

这个工具最初的想法只是一个网页内容提取和提示词可控的网页总结插件，加上创建时是vibe coding的早期，所以整个开发流程和框架没有设计（e.g. 没有使用任何现代化的前端开发框架）。后来增加了越来越多的功能，快速重构的失败彻底让位于对新功能的需要，最终就是一个代码不太好看但是好用的成品。

现在重构的想法逐渐变淡：一是重心转到思考认知提升流程进一步的问题，二是AI浏览器未来可能包含这些功能，三是希望vibe coding更加好用的时候，再重构将是一件事半功倍的事情。

## 核心功能
- 侧边栏调用，自动网页内容提取，自动多模型、多角度分析
- 历史页面管理，搜索
- 方便阅读和对话的细节功能
- 提示词的可控性

注意：需要自己配置大模型调用API

## 安装说明

### chrome商店
- https://chromewebstore.google.com/detail/think-bot/fnicniodcfoggafbmigcbdnjdcmfhpob

- 搜索”认知助手”或者“Think Bot”

### 本地安装

1. **克隆或下载项目**  

2. **在Chrome中加载扩展**
   - 打开Chrome浏览器
   - 进入 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目根目录

3. **配置大模型API密钥**
   - 点击扩展图标打开侧边栏
   - 进入设置页面
   - 配置所需的LLM模型API密钥
